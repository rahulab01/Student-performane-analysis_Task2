# 🚢 Titanic Data Analysis Project

## 📌 Overview
This project analyzes the Titanic dataset to understand survival patterns based on gender, passenger class, and age.

## 🎯 Objectives
- Data cleaning and preprocessing
- Survival analysis
- Data visualization

## 🛠️ Technologies Used
- Python
- Pandas
- Seaborn
- Matplotlib

## 📊 Key Insights
- Females had higher survival rates than males
- First-class passengers survived more
- Younger passengers had better chances of survival

## 📁 Files Included
- Titanic Analysis Notebook (.ipynb)
- Titanic Dataset (.csv)
